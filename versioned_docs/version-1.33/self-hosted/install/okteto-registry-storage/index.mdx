# Configure the Okteto Registry with external storage

Okteto supports different storage drivers to store the images inside the [Okteto Registry](core/container-registry.mdx):

- [Filesystem](filesystem.mdx)
- [Amazon S3 Bucket](aws-s3-bucket.mdx)
- [Azure Storage Container](azure-storage-container.mdx)
- [DigitalOcean Spaces](digitalocean-spaces.mdx)
- [Google Cloud Storage](google-cloud-storage.mdx)
