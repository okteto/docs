Okteto can be configured to automatically create endpoints for your application.
In order to do that, you need to add the following annotation to your [Ingress](https://github.com/okteto/movies-with-helm/blob/onboard-your-app/chart/templates/ingress.yaml) resource:

```yaml
annotations:
  dev.okteto.com/generate-host: "true"
```

This instructs Okteto to create an endpoint in the Okteto wildcard domain.
For example, if your development environment name is `movies` in the `cindylopez` namespace, it will be accessible at https://movies-cindylopez.okteto.example.com if your Okteto domain is okteto.example.com.
More information about how to configure Okteto Endpoints is available [on this link](core/ingress/automatic-ssl.mdx).
