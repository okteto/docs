This guide explains how to onboard your own applications to Okteto. This guide includes the following steps:

- Configure Okteto to deploy your application
- Expose your application using Okteto Endpoints
- Build your container images with Okteto Build
- Configure an Okteto Variable for a database password

> If you have a Docker Compose file, you can use it with Okteto with no changes. Follow [this guide](guides/tutorials/compose-getting-started.mdx) for more details.

## Requirements

- You have access to an Okteto instance. [Sign up for our free 30 days trial](https://www.okteto.com/free-trial/) to start the process
- You have [installed the Okteto CLI](get-started/install-okteto-cli.mdx/#installing-the-okteto-cli) and [configured the Okteto CLI](get-started/install-okteto-cli.mdx/#configuring-the-okteto-cli) with your Okteto instance
- You have completed our [getting started guide](get-started/deploy-first-env.mdx) and understand the basics of the Okteto development experience
