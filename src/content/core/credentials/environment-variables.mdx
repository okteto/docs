Environment variables provide another way to specify your credentials. This is the recommended method when scripting tasks or when [creating preview environments](previews/index.mdx).

The supported environment variables are:

- `OKTETO_TOKEN`: Specifies the [personal access token](core/credentials/personal-access-tokens.mdx) to use
- `OKTETO_CONTEXT`: Specifies the Okteto URL. If missing, it defaults to https://okteto.example.com
- `OKTETO_NAMESPACE`: Specifies the namespace to use. If missing, it defaults to the namespace of the current context

:::tip

You can use the `.env` file to store your credentials. See [Okteto Variables](/docs/core/okteto-variables) to know more.

:::
