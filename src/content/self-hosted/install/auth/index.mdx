There are numerous authentication methods available in Okteto to verify user identity.
The authentication configuration dictates which users can access Okteto and the method they can use for logging in.

Okteto provides OAuth2 integrations for the following auth providers:

- [Azure Active Directory](azure-ad.mdx)
- [Bitbucket](bitbucket.mdx)
- [GitHub](github.mdx)
- [GitLab](gitlab.mdx)
- [Google](google.mdx)
- [Okta](okta.mdx)
- [OpenID Connect](openid-connect.mdx)

> Are we missing your favorite identity provider? Open a feature request in the [Okteto Community](https://community.okteto.com/t/about-the-feature-requests-category/28) and let us know!

Additionally, Okteto provides a single user authentication method based on a [Token](token.mdx) for initial evaluations.
