Welcome! This guide is designed to help developers get started with Okteto, its concepts and terminology. Okteto enables you to deploy and develop applications directly in the cloud with a single [CLI command](https://www.okteto.com/docs/development/using-okteto-cli/) or click of a button. Write code locally on your machine and view your changes live, deployed in the cloud **as soon as you hit save**!

If you haven't done so yet, install and configure the Okteto CLI following [this guide](https://www.okteto.com/docs/get-started/install-okteto-cli/).

## What is Okteto and How Can I Benefit From it?

With Okteto, setting up your development environment is a one-click experience.
Featuring instantaneous Code Sync and Live Updates, experience the magic of seeing your code changes reflected instantly in your cloud environment.
This allows for rapid testing and iteration without the need for rebuilds or redeployments, ensuring that your applications run just as smoothly in development as they do in production.
